🌿 ForêtMap

L’interface utilisateur est désormais livrée par le build Vite (dossier dist/).

Si le serveur tourne en production (NODE_ENV=production) et que vous voyez cette page, vérifiez que le build frontend est présent dans dist/, puis redémarrez Node.

En développement : lancez le backend (ex. npm run dev sur le port 3000) et le client Vite (npx vite ou npm run dev:client) — voir docs/LOCAL_DEV.md.

Les assets publics (sw.js, images) restent dans public/ et sont copiés dans dist/ au build.
