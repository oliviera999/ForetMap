# ForetMap

Application de gestion de la forêt comestible — Lycée Lyautey.

- **Stack :** Node.js, Express, SQLite (better-sqlite3)
- **Démarrage :** `npm install` puis `npm start`

## Pousser ce dépôt sur GitHub

1. **Créer un nouveau dépôt sur GitHub**  
   Allez sur [github.com/new](https://github.com/new), choisissez un nom (ex. `ForetMap`), ne cochez pas « Initialize with README », puis cliquez sur **Create repository**.

2. **Ajouter le remote et pousser** (remplacez `VOTRE_UTILISATEUR` par votre nom d’utilisateur GitHub) :

   ```bash
   cd c:\projets_code\ForetMap
   git remote add origin https://github.com/VOTRE_UTILISATEUR/ForetMap.git
   git push -u origin main
   ```

Si vous préférez SSH :

```bash
git remote add origin git@github.com:VOTRE_UTILISATEUR/ForetMap.git
git push -u origin main
```
